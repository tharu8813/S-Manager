# S Manager
> ㅈ같은 핵쟁이와 학살, 방해로 부터 해방되세요!

## 사용법
<img width="451" height="635" alt="image" src="https://github.com/user-attachments/assets/172d42fb-5640-4e45-8359-495475a09267" />

1. 현재 설정된 상태를 표시합니다.
2. 설정된 게임 경로를 수정합니다.
3. 코드를 수정합니다. 복사, 붙여놓기와 현재 세션코드 가져오기를 할 수 있습니다. 또 자동 생성도 가능합니다.
4. 해당 코드로 세션을 생성하거나 공개 세션으로 변경합니다.

# 게임 경로

게임 경로를 게임을 어떤 런처로 다운받았는지에 따라 다릅니다. 또는 기본 경로가 아닌 다른 경로에 설치할 경우 다릅니다.

아래 경로는 런처마다 기본 설치경로가 나와있으니 참고하세요.

| Steam | `C:\Program Files (x86)\Steam\steamapps\common\Red Dead Redemption 2` |
| --- | --- |
| Epic Games Launcher | `C:\Program Files\Epic Games\Red Dead Redemption 2` |

# 친구와 함께하기

간단합니다

집에 주소가 있는데 그걸 설정한다고 생각하시면 됩니다.

같은 코드를 사용한 뒤 게임을 실행하면 됩니다. 그리고 온라인으로 접속하면 됩니다.

만약 안된다면 `L키` 를 눌러 세션에 플레이어가 있는지 확인해주세요. 유저가 나밖에 없다면 `Home키` 또는 친구가 세션 초대를 보낸뒤 수락해서 참가하시면 됩니다.

만약 다른 유저도 있다면 프로그램을 관리자 권한으로 실행수 다시시도해주세요.
